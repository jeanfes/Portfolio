Vibes

App de bienestar con sonidos ambientales y gestión de rutinas

Kotlin Android Jetpack Compose Room Database

#descripción

Vibes es una aplicación nativa de Android desarrollada en Kotlin que combina reproducción de sonidos ambientales con gestión inteligente de rutinas diarias. La app permite a los usuarios crear ambientes sonoros personalizados para meditación, concentración o relajación, mientras organizan sus tareas y hábitos con recordatorios programables.

Con una interfaz intuitiva desarrollada con Jetpack Compose y detección facial para personalizar la experiencia, Vibes ayuda a los usuarios a mejorar su productividad y bienestar mental mediante la combinación de audio inmersivo y organización personal. Todos los datos se almacenan localmente en el dispositivo usando Room Database, garantizando privacidad total.

#características-principales

  • Biblioteca de sonidos ambientales (lluvia, océano, bosque, fuego, etc.)
  • Mezcla personalizada de múltiples sonidos simultáneos
  • Detección facial para personalización de experiencia (procesamiento local)
  • Gestión de rutinas diarias con tareas organizadas por categorías
  • Sistema de recordatorios con notificaciones locales
  • Temporizador de sesiones de meditación o concentración
  • Modo oscuro nativo con diseño minimalista
  • Almacenamiento local sin conexión a internet requerida
  • Reproductor de audio con controles avanzados

#desafíos-y-soluciones

Detección Facial Local

Implementé ML Kit Face Detection de Google para procesar la detección facial completamente en el dispositivo sin enviar datos a servidores. Esto permite personalizar la experiencia del usuario mientras se mantiene la privacidad absoluta, procesando las imágenes en tiempo real sin almacenarlas.

Gestión de Audio Múltiple

Desarrollé un sistema robusto con ExoPlayer y MediaPlayer que permite mezclar múltiples sonidos simultáneamente con control independiente de volumen para cada pista, optimizando el uso de recursos del dispositivo y la duración de la batería mediante corrutinas de Kotlin.

Sistema de Notificaciones Inteligentes

Implementé WorkManager y AlarmManager para crear un sistema de programación de recordatorios que funciona incluso con la app cerrada, gestionando correctamente las restricciones de batería de Android y las políticas de Doze Mode para garantizar que las notificaciones lleguen en el momento correcto.

Arquitectura Limpia y Escalable

Utilicé Clean Architecture con MVVM, separando la lógica de negocio de la UI con Jetpack Compose. Implementé inyección de dependencias con Hilt y persisté datos con Room Database, creando una aplicación mantenible, testeable y fácil de escalar con nuevas funcionalidades.

#privacidad-y-datos

Vibes está diseñada con la privacidad como prioridad. No recopilamos ningún dato personal identificable y toda la información se almacena localmente en tu dispositivo. La aplicación funciona completamente offline y no requiere registro ni conexión a internet.
